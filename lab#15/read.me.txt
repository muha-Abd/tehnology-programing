# Лабораторная работа №15  
## Мини-проект: Менеджер задач

### Цель работы
Закрепить модульность, структуры данных, события, работу с API, локальное хранение данных, ООП, тестирование и отладку.

---

## Описание проекта
Приложение представляет собой простой менеджер задач.
Позволяет:
- добавлять задачи
- удалять
- хранить данные в LocalStorage
- отображать список в интерфейсе

Код разбит на 3 модуля:
- `main.js` — логика
- `storage.js` — хранение данных
- `ui.js` — интерфейс

Есть один unit-тест: проверка корректности сохранения/загрузки задач.
